  ИНСТРУКЦИЯ ДЛЯ ПРОВЕРКИ:
═══════════════════════════════════════════════════════════════

ШАГ 1: Откройте сайт на мобильном телефоне
   └─ URL: https://ecopackpro.ru
   └─ Используйте ЛЮБОЙ браузер (Chrome, Safari, Firefox)

ШАГ 2: Найдите любой товар
   └─ Перейдите в каталог
   └─ Выберите товар (например, "Воздушно пузырьковая пленка")

ШАГ 3: Добавьте товар в корзину
   └─ Нажмите кнопку "Добавить в корзину"

ШАГ 4: ПОСМОТРИТЕ НА САМЫЙ НИЗ ЭКРАНА! ⬇️
   └─ Там должна быть панель навигации:
   └─ [ Главная ] [ Каталог ] [ Избранное ] [ Кабинет ] [ Корзина ] [ Ещё ]

ШАГ 5: НА ЭЛЕМЕНТЕ "КОРЗИНА" ДОЛЖЕН ПОЯВИТЬСЯ:
   ✅ КРАСНЫЙ КРУГЛЫЙ BADGE (кружочек)
   ✅ С ЦИФРОЙ "1" (или количеством товаров)
   ✅ В ПРАВОМ ВЕРХНЕМ УГЛУ текста "Корзина"

   Примерно так:  Корзина ⓵
                         ↑
                    (красный кружок с цифрой)

ШАГ 6: Проверьте, что badge НЕ ИСЧЕЗАЕТ
   └─ Перейдите на главную страницу
   └─ Badge должен ОСТАВАТЬСЯ ВИДИМЫМ!
   └─ Количество должно быть ПРАВИЛЬНЫМ!

═══════════════════════════════════════════════════════════════
  ЧТО ЕСЛИ НЕ РАБОТАЕТ?
═══════════════════════════════════════════════════════════════

ВАРИАНТ 1: Очистите кэш браузера на телефоне
   └─ Настройки браузера → Очистить данные/кэш
   └─ Перезагрузите страницу

ВАРИАНТ 2: Откройте в режиме инкогнито
   └─ Это исключит проблемы с кэшем

ВАРИАНТ 3: Проверьте другой браузер
   └─ Попробуйте Chrome, если был Safari
   └─ Или наоборот

ВАРИАНТ 4: Перезагрузите страницу
   └─ Потяните страницу вниз для обновления
   └─ Или F5 / Ctrl+R

ВАРИАНТ 5: Проверьте консоль (для tech-savvy)
   └─ Chrome: chrome://inspect → Remote devices
   └─ Должны быть логи: [Mobile Footer Badge] Initializing...
